# ЯДРО СУБЪЕКТА · ЖИВОЙ ЦИКЛ

Стратегическая браузерная игра становления, собранная из архитектоники проекта «Гдея» и логики ТзАрх.

**Публичная версия:** `https://lacinnik.github.io/Game-GDEYA/`

Игра публикуется через GitHub Pages и открывается в Safari, Chrome и других браузерах без входа в ChatGPT. На iPhone её можно добавить на экран «Домой» из меню «Поделиться» Safari.

## Замысел

Игрок проводит живое намерение через четыре плиты:

**Ресурс → Власть → Отношения → Результат**

Цель — материализовать намерение, сохранив инвариант и не отдав ядро градиенту поля.

Каждый ход проходит четыре такта:

1. **Наблюдение** — различить точное число состояний, заданное картой.
2. **Конфигурация** — выбрать геометрию и тип точки.
3. **Формула** — собрать переход `Аз × Бука → Передача`.
4. **Отклик поля** — получить изменение метрик и фазы Источника.

## Что вошло в игру

- 96 карт: 4 плиты × 6 процессов × 4 фазы;
- быстрый цикл на 4 узла и полный канонический цикл на 24 узла;
- 5 геометрий, 4 типа точек, 49 Азов, 24 Буки и 7 Передач;
- автоматический расчёт `α`, `IЯ`, `Cₘ`, `Q` и `T`;
- слои Тело → Эмоции → Мысль → Творец;
- аварийный тактический контур и события `M⁷–M¹²`;
- итоговая когерентность и экспорт чертежа прохождения в JSON;
- голосовой проводник Скеллу: озвучивание шага, повтор инструкции и контекстная диктовка;
- адаптивный интерфейс с сенсорным управлением для iPhone.

## Локальный запуск

Требуется Node.js `22.13+`.

```bash
npm ci
npm run dev
```

После запуска откройте адрес, указанный Vite в терминале.

## Проверка

```bash
npm test
```

Команда собирает проект, валидирует артефакт и запускает тест отрисовки.
